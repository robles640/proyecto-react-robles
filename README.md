# Proyecto Final: Gestión de Tareas con React

Este proyecto es una aplicación funcional de "Lista de Tareas" (To-Do List) desarrollada en React. La aplicación permite gestionar el ciclo de vida completo de las tareas conectándose a una API externa.

## Funcionalidades Implementadas
- **Visualización**: Listado dinámico de tareas existentes con estilos visuales diferenciados según el estado.
- **Creación**: Formulario completo para añadir nuevas tareas a la base de datos.
- **Edición**: Capacidad de modificar el título o la descripción de tareas ya creadas.
- **Eliminación**: Opción para borrar tareas individuales de forma permanente.
- **Gestión de Estado**: Selector para alternar tareas entre estado "Pendiente" y "Finalizada".

## Tecnologías Utilizadas
- **React + Vite**: Entorno de desarrollo y librería principal.
- **Material UI (MUI)**: Framework de componentes para la interfaz de usuario.
- **Axios**: Cliente HTTP para el consumo de la API.
- **Zod**: Validación de esquemas para los datos de las tareas.

## Instrucciones para Correr el Proyecto
1. Clonar el repositorio.
2. Instalar dependencias con `npm install`.
3. Iniciar el servidor de desarrollo con `npm run dev`.
4. Para desplegar, usar `npm run deploy`.
